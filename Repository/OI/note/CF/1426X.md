## A. Floor Number

特判一下$n \leq 2$的情况，输出`(n + x - 3) / x + 1`即可

## B. Symmetric Matrix

只要有一个能放到对角线上的矩形和一组对称的矩形即可

## C. Increase and Copy

求一个单调上升的谷堆的左上投影长度的最小值，显然根号最优，模拟每次去贴皮即可

## D. Increase and Copy

`Tian_Xing`和`qwaszx`都认为这个题比E和F难，但我和`e老师`都认为这个题简单

考虑一段和为$0$的序列在前缀和上表现为两个相等的数中间的一段，离散化后找出这些序列，然后贪心切即可
