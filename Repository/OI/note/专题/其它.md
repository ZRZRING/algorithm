#### 字符串操作-substr函数

$substr(i,len)$返回下标$i$开始共$len$位字符

$substr(i)$返回下标$i$位开始所有字符

举个栗子，假设一个字符串：$abcdefghig$

$substr(5,3)=fgh$，$substr(5)=fghig$

#### 离散化

```cpp
for (int i=1;i<=n;i++){
    a[i]=read();b[i]=a[i];
}
sort(b+1,b+n+1);
cnt=unique(b+1,b+n+1)-b-1;//记录b的长度
for (int i=1;i<=n;i++){
    a[i]=lower_bound(b+1,b+cnt+1,a[i])-b;
}
```
